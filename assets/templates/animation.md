# 动画分镜模板

## 2D/3D动画短片模板 (2-3分钟)

**帧率**: 24fps
**总镜头数**: 25-40个

### 格式说明
动画分镜需要额外标注关键姿势(KEY)、分解帧(BRK)和口型同步。

## 动画分镜特殊标注

### 口型同步标注
```
对白: "我不会放弃"
口型: REST → B/U → O → REST → A → I → A → REST
```

### 预备动作标注
```
KEY-1: 站立（预备姿势）
  ↓ [ANTICIPATION: 先微微下蹲]
KEY-2: 跳跃（主动作）
  ↓ [FOLLOW THROUGH: 衣服/头发滞后飘动]
KEY-3: 落地（结束姿势）
  ↓ [SQUASH: 落地挤压]
KEY-4: 恢复站立
```

### 挤压与拉伸标注
```
KEY-1: 正常站立（体积V=100%）
KEY-2: 起跳拉伸（高度+30%，宽度-15%，体积≈100%）
KEY-3: 落地挤压（高度-20%，宽度+25%，体积≈100%）
```

## 不同动画风格调整

| 风格 | 帧率 | 帧数调整 | 特殊注意 |
|------|------|----------|----------|
| 日式动画 | 24fps | 一拍二为主(12f/s) | 静止帧多，表情特写多 |
| 迪士尼/欧美 | 24fps | 一拍一(24f/s) | 动作流畅，预备动作充分 |
| Motion Graphics | 30fps | 全帧 | 缓动曲线(Easing)标注 |
| Flash/有限动画 | 24fps | 一拍二/三 | 节省帧数，强调构图 |
