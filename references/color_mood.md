# 色彩与影调设计指南

## 色彩在分镜中的角色

**色彩不是"好看"的问题——它是叙事工具。**

| 功能 | 说明 | 示例 |
|------|------|------|
| **情绪传递** | 色彩直接触发情绪反应 | 红色→激情/危险；蓝色→冷静/忧郁 |
| **时间暗示** | 色温暗示时间段 | 暖色→黄昏/室内；冷色→清晨/户外 |
| **空间区分** | 不同场景用不同色调 | 办公室冷白 vs 家中暖黄 |
| **角色标识** | 角色专属色彩 | 《黑客帝国》绿色=矩阵 |
| **主题隐喻** | 色彩承载主题意义 | 《辛德勒的名单》红衣=生命/希望 |
| **风格统一** | 全片色调建立视觉风格 | 韦斯·安德森的粉彩世界 |

## 色彩心理学基础

### 暖色系

| 色彩 | 正面情绪 | 负面情绪 | 典型应用 |
|------|----------|----------|----------|
| **红色** | 激情、爱、力量 | 危险、愤怒、血腥 | 浪漫、警告、暴力 |
| **橙色** | 温暖、活力、创意 | 焦虑、廉价感 | 家庭、秋季、食物 |
| **黄色** | 快乐、希望、阳光 | 警告、疾病、背叛 | 儿童、阳光、警示 |
| **粉色** | 浪漫、温柔、纯真 | 幼稚、脆弱 | 爱情、少女、梦境 |

### 冷色系

| 色彩 | 正面情绪 | 负面情绪 | 典型应用 |
|------|----------|----------|----------|
| **蓝色** | 冷静、信任、专业 | 忧郁、冷漠、疏离 | 企业、科技、孤独 |
| **绿色** | 自然、生命、平和 | 嫉妒、毒素、腐烂 | 自然、医疗、诡异 |
| **紫色** | 神秘、高贵、创意 | 颓废、不安 | 奇幻、皇室、心理 |
| **青色** | 清新、现代、冷静 | 冷漠、人工感 | 科技、未来、冷峻 |

### 中性色

| 色彩 | 情绪 | 典型应用 |
|------|------|----------|
| **黑色** | 权力、神秘、死亡 | 黑色电影、反派、正式 |
| **白色** | 纯洁、简约、死亡 | 婚礼、医院、天堂 |
| **灰色** | 中性、压抑、现实 | 纪录片、压抑环境 |
| **棕色** | 稳重、自然、怀旧 | 复古、西部、历史 |

## 影调类型

### 按亮度分布

| 影调 | 定义 | 情绪 | 典型应用 |
|------|------|------|----------|
| **高调 (High Key)** | 画面以亮部为主，暗部少 | 明亮、乐观、开放 | 喜剧、广告、浪漫 |
| **低调 (Low Key)** | 画面以暗部为主，亮部少 | 神秘、压抑、危险 | 悬疑、黑色电影、恐怖 |
| **中间调 (Mid Key)** | 亮暗均衡 | 中性、自然 | 纪录片、日常场景 |

### 按对比度

| 影调 | 定义 | 情绪 | 典型应用 |
|------|------|------|----------|
| **软调 (Soft)** | 对比度低，过渡柔和 | 温柔、梦幻、怀旧 | 浪漫、回忆、儿童 |
| **硬调 (Hard)** | 对比度高，明暗分明 | 强烈、戏剧、冲击 | 黑色电影、动作、冲突 |
| **平调 (Flat)** | 对比度极低，接近单色 | 压抑、疏离、现代 | 科幻、艺术片 |

## 经典色彩方案

### 1. 单色方案 (Monochromatic)

**原理**：单一色相，通过明度/饱和度变化创造层次。

**效果**：统一、和谐、专注。

**典型应用**：
- 《黑客帝国》：绿色单色（矩阵世界）
- 《辛德勒的名单》：黑白单色（历史感）

**分镜标注**：
```
色彩方案: 单色-蓝色
主色: #1a3a5c (深蓝)
辅色: #4a7a9c (中蓝)
点缀: #8ab4d8 (浅蓝)
情绪: 冷峻、科技、压抑
```

---

### 2. 互补色方案 (Complementary)

**原理**：色轮上相对的两种颜色（如蓝-橙、红-青）。

**效果**：对比强烈、视觉冲击、活力。

**典型应用**：
- 《疯狂麦克斯》：橙黄沙漠 + 青蓝天空
- 大多数动作片：暖色主体 + 冷色背景

**分镜标注**：
```
色彩方案: 互补色-橙蓝
主色: #e67e22 (橙色)
互补: #2980b9 (蓝色)
比例: 70%暖色主体 / 30%冷色背景
情绪: 冲突、活力、戏剧性
```

---

### 3. 类似色方案 (Analogous)

**原理**：色轮上相邻的2-3种颜色。

**效果**：和谐、自然、舒适。

**典型应用**：
- 日落场景：红-橙-黄渐变
- 森林场景：黄绿-绿-青绿

**分镜标注**：
```
色彩方案: 类似色-暖色渐变
色相: 黄(60°) → 橙(30°) → 红(0°)
情绪: 温暖、自然、时间流逝
应用: 黄昏场景，从黄色渐变到红色
```

---

### 4. 三色方案 (Triadic)

**原理**：色轮上等距的三种颜色（如红-黄-蓝）。

**效果**：丰富、活泼、平衡。

**典型应用**：
- 韦斯·安德森电影：精心设计的三色搭配
- 儿童内容：明亮的三原色

**分镜标注**：
```
色彩方案: 三色-韦斯安德森风格
色相: 粉红(330°) / 黄(60°) / 青(180°)
饱和度: 高饱和，统一调性
情绪: 童饰、童话、精心设计
```

---

### 5. 分裂互补 (Split Complementary)

**原理**：一个主色 + 互补色两侧的两个颜色。

**效果**：比互补色柔和，但仍有对比。

**典型应用**：
- 商业广告：主色品牌色 + 分裂互补作为点缀

## 色彩叙事技巧

### 技巧1：角色专属色彩

| 角色 | 专属色 | 含义 | 变化 |
|------|--------|------|------|
| 主角A | 蓝色 | 冷静、理性 | 变暖→情感觉醒 |
| 主角B | 红色 | 激情、冲动 | 变暗→堕落 |
| 反派 | 黑/紫 | 权力、威胁 | 始终统一 |
| 希望 | 白/金 | 救赎、光明 | 仅在关键时刻出现 |

**分镜应用**：
```
镜号: 15
画面: 主角A穿蓝色外套
色彩叙事: 蓝色=理性状态
备注: 后续场景蓝色渐暖，暗示情感觉醒
```

---

### 技巧2：场景色彩区分

| 场景 | 色调 | 情绪 | 光源 |
|------|------|------|------|
| 办公室 | 冷白 | 压抑、机械 | 日光灯 |
| 家中 | 暖黄 | 温暖、安全 | 白炽灯 |
| 夜店 | 霓虹 | 混乱、诱惑 | 彩色灯 |
| 医院 | 冷绿 | 冰冷、疏离 | 手术灯 |
| 战场 | 沙黄+灰 | 残酷、真实 | 自然光 |

**分镜应用**：
```
场景色彩设计:
  场景A (办公室): 冷白 #d0d5dd，低饱和，硬调
  场景B (家中): 暖黄 #f5d76e，高饱和，软调
  转场: 从A到B，色温从5600K→3200K
```

---

### 技巧3：时间线色彩变化

**《末代皇帝》色彩时间线**：

| 时期 | 色彩 | 含义 |
|------|------|------|
| 紫禁城童年 | 暖黄、金色 | 皇权、封闭 |
| 天津/东北 | 灰色、冷色 | 失去权力、迷茫 |
| 战犯管理所 | 灰白 | 现实、改造 |
| 文革 | 红色 | 动荡、危险 |
| 晚年 | 柔和暖色 | 平静、和解 |

**分镜应用**：
```
色彩时间线:
  第一幕 (童年): 暖黄主调，金色点缀
  第二幕 (青年): 逐渐变冷，增加灰色
  第三幕 (中年): 冷灰主调，低饱和
  第四幕 (晚年): 回归暖色，柔和
备注: 色彩变化与角色命运同步
```

---

### 技巧4：情绪转折的色彩变化

**原理**：在情绪转折点改变画面色彩。

| 转折类型 | 色彩变化 | 示例 |
|----------|----------|------|
| 希望→绝望 | 暖→冷，饱和度降低 | 《肖申克的救赎》多次 |
| 现实→回忆 | 自然色→偏暖/偏黄 | 《泰坦尼克号》 |
| 安全→危险 | 正常→偏红/偏绿 | 恐怖片 |
| 混乱→平静 | 高对比→低对比 | 战场→宁静 |
| 孤独→连接 | 冷→暖 | 浪漫片 |

---

### 技巧5：象征性色彩

| 色彩 | 经典象征 | 案例 |
|------|----------|------|
| **红色** | 生命/危险/爱 | 《辛德勒的名单》红衣女孩 |
| **绿色** | 嫉妒/自然/希望 | 《了不起的盖茨比》绿光 |
| **白色** | 纯洁/死亡/虚无 | 《教父》橙园场景 |
| **黑色** | 死亡/权力/未知 | 黑色电影 |
| **金色** | 权力/财富/神圣 | 皇室/宗教场景 |

## 导演色彩风格

### 韦斯·安德森 (Wes Anderson)

**特点**：
- 高饱和度粉彩
- 精确的对称构图配合色彩对称
- 每部电影有专属色板

**色板示例**：
| 电影 | 主色板 |
|------|--------|
| 《布达佩斯大饭店》 | 粉红、紫、淡蓝、金色 |
| 《月升王国》 | 黄绿、橙、暖黄 |
| 《水中生活》 | 海蓝、白、橙 |

---

### 大卫·芬奇 (David Fincher)

**特点**：
- 冷色调为主（青绿偏色）
- 高对比度
- 阴影区域偏青，高光偏橙

**调色方向**：
```
阴影: 偏青绿 #1a2a2a
中间调: 偏冷灰 #4a5a5a
高光: 偏暖橙 #d4a574
对比度: 高
饱和度: 中低
```

---

### 王家卫

**特点**：
- 高饱和度
- 霓虹色彩
- 暖色偏红，冷色偏绿

**典型场景**：
| 电影 | 场景 | 色彩 |
|------|------|------|
| 《花样年华》 | 走廊 | 暖红、暗红 |
| 《重庆森林》 | 夜市 | 霓虹绿、霓虹红 |
| 《堕落天使》 | 酒吧 | 冷蓝、霓虹 |

---

### 丹尼斯·维伦纽瓦 (Denis Villeneuve)

**特点**：
- 低饱和度
- 单色倾向
- 橙青互补色

**色板**：
```
《银翼杀手2049》:
  主色: 橙黄 #d4a574
  辅色: 青蓝 #1a3a4a
  饱和度: 低
  对比度: 高

《降临》:
  主色: 冷灰绿
  饱和度: 极低
  情绪: 冷峻、未知
```

## 分镜色彩标注格式

### 简洁版

```
色彩: 暖色调，黄昏氛围
影调: 中间调，软对比
```

### 详细版

```
═══════════════════════════════════════
色彩设计
═══════════════════════════════════════
方案: 互补色-橙蓝
主色: #e67e22 (橙色) - 主体
辅色: #2980b9 (蓝色) - 背景/阴影
点缀: #f1c40f (黄色) - 关键道具

影调: 低调，硬对比
色温: 3200K (暖)
饱和度: 中等

情绪目标: 冲突、戏剧性
叙事功能: 橙色=危险/激情，蓝色=冷静/理性

光源:
  主光: 橙色侧光 (夕阳)
  环境光: 蓝色补光 (天空)

备注: 后续场景橙色渐弱，蓝色渐强
═══════════════════════════════════════
```

## 不同视频类型的色彩策略

| 类型 | 色彩策略 | 影调 |
|------|----------|------|
| **广告** | 品牌色为主，高饱和，高调 | 明亮、积极 |
| **剧情片** | 服务叙事，可变化 | 随情绪变化 |
| **恐怖片** | 冷色、低饱和、低调 | 压抑、不安 |
| **喜剧** | 暖色、高饱和、高调 | 明亮、轻松 |
| **科幻** | 冷色、单色倾向、硬对比 | 未来感、疏离 |
| **纪录片** | 自然色、中间调 | 客观、真实 |
| **MV** | 风格化，可极端 | 视觉冲击 |
| **短视频** | 高饱和、明亮 | 吸引注意力 |

## 色彩设计检查清单

### 基础检查
- [ ] 全片色调是否统一？
- [ ] 不同场景是否有色彩区分？
- [ ] 色彩是否服务叙事（不只是好看）？
- [ ] 光源逻辑是否合理？

### 叙事检查
- [ ] 角色是否有专属色彩？
- [ ] 情绪转折是否有色彩变化？
- [ ] 是否有象征性色彩？
- [ ] 色彩变化是否与角色命运同步？

### 技术检查
- [ ] 色彩方案是否可执行（灯光/调色）？
- [ ] 不同场景的色彩过渡是否自然？
- [ ] 是否考虑了不同显示设备的色彩表现？
- [ ] 是否有色彩参考图？
