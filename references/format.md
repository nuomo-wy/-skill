# 分镜格式规范

## 标准分镜表格式

### 基础版（快速沟通）
| 镜号 | 景别 | 画面内容 | 台词/声音 | 时长 |
|------|------|----------|-----------|------|
| 1 | 全景 | 城市夜景，霓虹闪烁 | 城市环境音 | 3s |
| 2 | 中景 | 主角站在天台边缘 | 风声 | 4s |

### 完整版（制作使用）
| 镜号 | 景别 | 机位 | 运镜 | 画面内容 | 声音 | 时长 | 备注 |
|------|------|------|------|----------|------|------|------|
| 1 | 远景 | 俯拍 | 固定 | 城市全景，灯火通明 | 音乐起 | 5s | 航拍 |
| 2 | 中景 | 平拍 | 推 | 主角背影走向天台 | 脚步声 | 4s | 斯坦尼康 |

### 专业版（含技术参数）
```
镜号: 001
景别: 中近景
焦距: 50mm
机位: 平拍
运镜: 固定→缓推
画面: 主角面部，从迷茫到坚定的表情变化
声音: 内心独白"我必须这样做"
时长: 6s
灯光: 侧光，暖调
备注: 需要浅景深，背景虚化
```

## 分镜脚本格式

### 剧本式
```
场景：天台 - 夜晚

镜头1 远景 俯拍
城市夜景，灯火如繁星。
音乐起，钢琴独奏。
【5秒】

镜头2 中景 平拍 推
主角站在天台边缘，风吹动衣角。
缓缓推近，表情逐渐清晰。
【4秒】

镜头3 特写
主角眼神，从迷茫到坚定。
内心独白：我必须这样做。
【3秒】
```

## 景别标注规范

### 中文标注
- 远景 (远)
- 全景 (全)
- 中全景 (大全)
- 中景 (中)
- 中近景 (中近)
- 近景 (近)
- 特写 (特)
- 大特写 (大特)

### 英文标注
- ELS (Extreme Long Shot)
- LS (Long Shot)
- MLS (Medium Long Shot)
- MS (Medium Shot)
- MCU (Medium Close Up)
- CU (Close Up)
- ECU (Extreme Close Up)

## 运镜标注规范

### 基础运镜
| 中文 | 英文 | 缩写 |
|------|------|------|
| 固定 | Static | STAT |
| 推 | Dolly In / Push In | PUSH |
| 拉 | Dolly Out / Pull Out | PULL |
| 摇 | Pan | PAN |
| 移 | Truck / Dolly | TRUCK |
| 跟 | Tracking | TRACK |
| 升 | Crane Up | UP |
| 降 | Crane Down | DOWN |
| 手持 | Handheld | HAND |
| 变焦 | Zoom | ZOOM |

### 复合运镜
- 推+摇：PUSH PAN
- 移+升：TRUCK UP
- 跟+拉：TRACK PULL

## 机位标注

### 水平角度
- 正面 (Front)
- 侧面 (Side)
- 斜侧 (3/4)
- 背面 (Back)

### 垂直角度
- 俯拍 (High Angle / Bird's Eye)
- 平拍 (Eye Level)
- 仰拍 (Low Angle / Worm's Eye)

### 综合标注示例
- 俯拍正面：HIGH FRONT
- 平拍斜侧：EYE 3/4
- 仰拍背面：LOW BACK

## 时长计算

### 基础公式
总时长 = 各镜头时长之和 + 转场时间

### 广告时长分配
- 15秒：8-12个镜头
- 30秒：12-20个镜头
- 60秒：20-35个镜头

### 剧情片参考
- 对话场景：3-8秒/镜头
- 动作场景：1-3秒/镜头
- 情感场景：5-10秒/镜头

## 分镜输出格式选择

### 选择表格格式当：
- 需要清晰对比各镜头
- 制作团队需要快速查阅
- 镜头数量较多

### 选择剧本格式当：
- 强调叙事 flow
- 需要详细描述画面
- 与编剧/导演沟通

### 选择可视化格式当：
- 向非专业人员展示
- 需要直观感受
- 提案/ pitching 使用
