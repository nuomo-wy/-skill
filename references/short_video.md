# 短视频与竖屏分镜指南

## 竖屏视频的独特性

### 画幅差异
| 维度 | 横屏 16:9 | 竖屏 9:16 |
|------|-----------|-----------|
| 视野 | 宽阔，环境信息多 | 窄，聚焦主体 |
| 人像 | 全身，环境丰富 | 半身/头像为主 |
| 横向运动 | 自然 | 受限 |
| 纵向运动 | 受限 | 自然 |
| 信息布局 | 分散在宽度上 | 集中在高度上 |
| 字幕区 | 底部安全区 | 底部1/4为字幕区 |

### 竖屏构图原则
- **主体居中或偏上**：避免底部被字幕/UI遮挡
- **纵向引导线**：利用建筑/道路的纵向线条
- **分层构图**：前/中/后景纵向排列
- **紧凑画框**：避免过多留白（竖屏留白更显空洞）
- **特写优先**：竖屏天生适合面部特写

## 短视频分镜核心原则

### 前3秒法则
**前3秒决定生死**，必须完成：
1. **抓住注意力**：视觉冲击/悬念/问题/冲突
2. **传递内容类型**：让观众知道这是什么视频
3. **建立观看动机**：为什么要继续看

### 短视频节奏控制

| 平台 | 常见时长 | 节奏特点 |
|------|----------|----------|
| 抖音/TikTok | 15-60秒 | 极快，3-5秒换一次视觉 |
| 快手 | 15-180秒 | 稍慢，注重内容 |
| 视频号 | 30-300秒 | 中等，社交驱动 |
| 小红书 | 30-180秒 | 中等，精美 |
| Instagram Reels | 15-90秒 | 快节奏，视觉驱动 |
| YouTube Shorts | 15-60秒 | 中等偏快 |
| B站竖屏 | 30-300秒 | 内容驱动，可慢 |

### 短视频分镜结构

#### 15秒超短视频
```
0-2s   钩子（视觉冲击/悬念/问题）
2-8s   核心内容展示
8-13s  高潮/反转
13-15s 结尾/CTA
```

#### 30秒标准短视频
```
0-3s   钩子（抓住注意力）
3-10s  内容展开（建立兴趣）
10-20s 核心信息/高潮
20-25s 补充/反转
25-30s 结尾/CTA/互动引导
```

#### 60秒深度短视频
```
0-3s   钩子
3-15s  背景铺垫/问题建立
15-40s 核心内容/解决方案
40-50s 补充信息/案例
50-60s 总结/CTA
```

## 不同类型短视频分镜

### 1. 知识科普类

**分镜要点**：
- 开头抛出问题/悬念
- 信息分层递进
- 视觉辅助理解（图表/动画/实景）
- 结尾总结要点

**分镜模板**：
```
镜头1 [近景] 主角直视镜头抛问题 "你知道XX吗？"
镜头2 [中景] 主角走入场景/展示画面
镜头3-N [特写/中景] 信息讲解（配合字幕/图示）
镜头N+1 [近景] 总结 "记住这3点..."
镜头N+2 [近景] CTA "关注我，下期讲..."
```

### 2. 好物推荐类

**分镜要点**：
- 开头展示产品最吸引人的画面
- 使用前后对比
- 使用场景化展示
- 特写质感

**分镜模板**：
```
镜头1 [特写] 产品最吸引人的画面（质感/效果）
镜头2 [近景] "这个XX真的绝了"
镜头3 [中景] 开箱/展示过程
镜头4 [特写] 产品细节
镜头5 [中景] 使用场景（自然融入）
镜头6 [特写] 使用效果对比
镜头7 [近景] 总结 "这个XX推荐给..."
镜头8 [近景] CTA
```

### 3. 情景剧情类

**分镜要点**：
- 快速建立角色和情境
- 冲突在5秒内出现
- 反转或笑点在15-20秒
- 表情特写关键

**分镜模板**：
```
镜头1 [中景] 场景建立（办公室/教室/家庭）
镜头2 [近景] 角色A面临问题
镜头3 [近景] 角色B反应/冲突
镜头4 [特写] 关键道具/表情
镜头5 [中景] 高潮/反转
镜头6 [近景] 角色反应（搞笑/感动）
镜头7 [近景] 结尾金句/互动引导
```

### 4. Vlog/日常类

**分镜要点**：
- 真实感优先
- 旁白驱动叙事
- 画面节奏与旁白匹配
- 美感+日常平衡

**分镜模板**：
```
镜头1 [特写] 美食/场景特写（视觉钩子）
镜头2 [近景] 主角面对镜头 "今天..."
镜头3-N [混剪] 日常场景（做饭/逛街/工作）
旁白贯穿
镜头N+1 [中景] 反思/感悟
镜头N+2 [近景] 结尾/互动
```

### 5. 教程/教程类

**分镜要点**：
- 每一步清晰展示
- 手部操作特写
- 结果对比明显
- 步骤编号/字幕

**分镜模板**：
```
镜头1 [近景] "3步搞定XX"
镜头2 [特写] 第一步操作
镜头3 [特写] 第一步结果
镜头4 [特写] 第二步操作
镜头5 [特写] 第二步结果
镜头6 [特写] 第三步操作
镜头7 [中景] 最终效果展示
镜头8 [近景] "关注我看更多"
```

## 短视频运镜技巧

### 适合竖屏的运镜

| 运镜 | 效果 | 注意事项 |
|------|------|----------|
| **固定** | 稳定、干净 | 短视频最常用 |
| **推** | 聚焦、强调 | 缓推效果好 |
| **拉** | 揭示、结束 | 适合场景转换 |
| **横摇** | 展示宽度受限空间 | 幅度不宜大 |
| **升降** | 从脚到头/反之 | 天然适合竖屏 |
| **手持** | 真实、日常 | Vlog常用 |
| **变焦快推** | 冲击、搞笑 | 短视频标志性手法 |

### 短视频专属手法

**画面跳跃 (Jump Cut)**
- 连续拍摄同一主体
- 剪掉中间停顿
- 制造时间快进效果
- 短视频最常见剪辑手法

**一镜到底 + 分屏**
- 单一镜头走完整个场景
- 快速切换不同区域
- TikTok 热门手法

**速度变化**
- 正常→慢动作（强调）
- 慢动作→正常（释放）
- 快速加速（搞笑/过渡）

## 短视频字幕与UI

### 字幕规范
- **位置**：底部 1/4 区域
- **字号**：画面高度的 4-6%
- **安全区**：距左右边缘各 10%，距底部 15%
- **每行字数**：不超过 15 个中文字符
- **停留时间**：每行至少 1.5 秒

### 互动元素
- **点赞/评论引导**：结尾 2-3 秒
- **表情贴纸**：增强情感表达
- **进度条/倒计时**：制造期待
- **指向箭头/文字**：引导关注

## 短视频分镜格式

### 快速分镜表
```
| 镜号 | 时长 | 画面描述 | 景别 | 运镜 | 声音/旁白 | 字幕 | 备注 |
|------|------|----------|------|------|-----------|------|------|
| 1 | 2s | 主角抛问题 | 近景 | 固定 | "你知道吗？" | 你知道吗？ | 直视镜头 |
```

### 详细分镜
```
镜号: 01
时长: 00:00 - 00:02
画面: [近景 固定] 主角直视镜头，表情惊讶
运镜: 固定
旁白: "你知道90%的人都不会用这个功能吗？"
字幕: 90%的人都不知道的功能
音乐: 悬疑音效起
备注: 眼神直视镜头，制造对话感
```

## 短视频分镜检查清单

### 开头（0-3秒）
- [ ] 第一帧就能抓住注意力？
- [ ] 前3秒传递了什么信息？
- [ ] 观众有理由继续看吗？

### 内容（3秒-结束前）
- [ ] 信息传达清晰？
- [ ] 节奏是否有变化？
- [ ] 视觉是否有新鲜感？
- [ ] 字幕清晰可读？

### 结尾（最后3秒）
- [ ] 有明确的CTA？
- [ ] 留下记忆点？
- [ ] 引导互动（点赞/评论/关注）？

### 整体
- [ ] 适合竖屏观看？
- [ ] 无声观看也能理解？
- [ ] 时长是否合理（不拖沓）？
- [ ] 前3秒能让人停下来？
