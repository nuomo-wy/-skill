# 动画分镜完全指南

## 动画分镜与实拍分镜的区别

| 维度 | 动画分镜 | 实拍分镜 |
|------|----------|----------|
| 自由度 | 完全自由（不受物理限制） | 受物理/场地/预算限制 |
| 精确度 | 必须精确到每一帧 | 拍摄时可以调整 |
| 时间标注 | 精确到帧 | 通常精确到秒 |
| 关键帧 | 必须标注动作关键姿势 | 不需要 |
| 摄影机 | 可描述任何运动 | 受设备限制 |
| 成本影响 | 分镜直接决定工作量 | 分镜是参考，不是最终 |
| 预演 | 分镜本身就是预演 | 需要额外制作预演 |

## 动画分镜核心要素

### 1. 关键姿势 (Key Pose / Key Frame)
**定义**：动作中最具表现力的姿势

**标注方式**：
```
[关键帧 A] 角色站立，双手自然下垂
    ↓ 动作描述：缓慢抬起右手
[关键帧 B] 角色右手举过头顶，表情惊讶
```

**关键帧类型**：
| 类型 | 缩写 | 说明 |
|------|------|------|
| **关键帧** | KEY | 动作起点/终点/转折点 |
| **分解帧** | BRK | 关键帧之间的过渡姿势 |
| **中间帧** | INB | 自动生成的插值帧 |

### 2. 口型同步 (Lip Sync)
**标准口型**：
| 发音类型 | 口型描述 | 对应发音 |
|----------|----------|----------|
| A | 嘴巴大张 | a, ah |
| B/M/P | 双唇紧闭 | b, m, p |
| C/D/G/K | 舌抵上颚 | c, d, g, k |
| E | 嘴角横向拉开 | e |
| F/V | 上齿咬下唇 | f, v |
| L | 舌尖顶上颚 | l |
| O | 嘴巴圆形 | o, oh |
| REST | 闭嘴/自然 | 静默 |

**标注方式**：
```
旁白: "我必须这样做"
口型: REST → A → M → U → J → I → J → YAO → ZHE → YANG → ZUO → REST
```

### 3. 动作线 / 动线 (Action Line)
**定义**：角色运动的轨迹线

**标注**：
```
┌──────────────────┐
│                  │
│  角色A           │
│    ↘             │  ← 动线：斜向下移动
│      角色B       │
│                  │
└──────────────────┘
```

### 4. 预备动作 (Anticipation)
**原则**：任何主要动作之前必须有反向预备动作

**示例**：
```
想要向上跳 → 先向下蹲（预备）
想要向右跑 → 先向左倾（预备）
想要快速击打 → 先收回拳头（预备）
```

### 5. 挤压与拉伸 (Squash & Stretch)
**原则**：角色/物体运动中保持体积不变的前提下变形

**效果**：
- 挤压：落地瞬间变扁
- 拉伸：跳跃时拉长
- 增强重量感和弹性感

## 动画分镜流程

### 阶段1：剧本 → 分镜描述
```
剧本文字 → 逐镜头的文字描述 → 标注关键动作和情绪
```

### 阶段2：粗分镜 (Rough Storyboard)
```
快速草图 → 基本构图 → 关键姿势 → 粗略时间标注
```
- 不追求画面精美
- 强调构图和叙事
- 快速迭代

### 阶段3：精细分镜 (Clean Storyboard)
```
细化构图 → 完善背景 → 标注所有技术参数
```
- 更精确的画面
- 完整的背景
- 镜头运动标注
- 声音标注

### 阶段4：动态分镜 (Animatic)
```
分镜图 → 按时间排列 → 配合临时音效/音乐 → 确定节奏
```
- 这是动画制作的关键里程碑
- 所有时间在此确定
- 需要导演最终批准

### 阶段5：布局 (Layout)
```
动态分镜 → 精确背景 → 精确角色位置 → 摄影机参数
```

## 动画分镜标注格式

### 标准标注
```
场景: SC-03
镜头: SH-015
时长: 3秒12帧 (84 frames)
帧率: 24fps

画面描述:
[中景] 角色A站在窗前，转身面向镜头
表情从忧郁变为坚定

关键姿势:
KEY-1 (0f):   侧身面向窗户，肩膀低垂
BRK-1  (36f): 开始转身
KEY-2  (48f): 面向镜头，眼神坚定，微微扬下巴

摄影机:
固定镜头，无运动

声音:
环境音: 风声渐弱
音效: 脚步声（转身时）
BGM: 弦乐渐强

备注:
需要强调表情变化
嘴唇微动（无声独白）
```

## 不同动画类型的分镜

### 2D 传统动画
**分镜特点**：
- 强调角色表演（夸张表情和动作）
- 逐帧标注关键姿势
- 背景分层（远/中/近）
- 颜色标注指定关键元素

**代表**：迪士尼经典动画（《狮子王》《美女与野兽》）

**分镜要点**：
- 角色原型设计先于分镜
- 每个关键姿势都要有图纸编号
- 动画12法则贯穿始终

---

### 3D 电脑动画
**分镜特点**：
- 分镜→布局→动画的严格流程
- 3D虚拟摄影机完全自由
- 灯光/材质在后期确定
- 预演（Previs）是关键环节

**代表**：皮克斯（《玩具总动员》《飞屋环游记》）、梦工厂

**分镜要点**：
- 预演制作比传统动画更重要
- 需要标注摄影机参数（焦距/景深/运动曲线）
- 虚拟摄影可参考实拍电影技法
- 与灯光/特效团队紧密协作

---

### 日式动画 (Anime)
**分镜特点**：
- 有限动画理念（减少帧数，强调镜头语言）
- 固定镜头大量使用
- 特写/远景的极端对比
- "静止的表演"（表情/眼神变化代替动作）

**代表**：宫崎骏（《千与千寻》）、今敏（《红辣椒》）、新海诚（《你的名字》）

**分镜要点**：
- 摄影表（X-sheet）精确到帧
- 大量使用"一拍二"（每帧拍两次）
- 背景美术质量极高
- 分镜画稿本身就是艺术品

**日式动画分镜特有标注**：
```
カット: 15A (Cut 15A)
尺: 3秒 (72 frames) ← 24fps
```

---

### 动态图形 (Motion Graphics)
**分镜特点**：
- 文字/图形动画为主
- 时间轴精确控制
- 与音乐节奏严格匹配
- 风格统一是核心

**代表**：MV片头、APP展示、品牌动画

**分镜要点**：
- 每个元素的进入/退出时机
- 动画缓动曲线（Easing）标注
- 颜色变化时间点
- 文字出现方式（打字/淡入/滑动）

## 动画12法则与分镜

由迪士尼动画师 Frank Thomas 和 Ollie Johnston 总结：

| 法则 | 分镜中的应用 |
|------|-------------|
| **Squash & Stretch** | 标注角色落地/跳跃的形变 |
| **Anticipation** | 在关键动作前标注预备帧 |
| **Staging** | 构图确保主体清晰 |
| **Straight Ahead / Pose to Pose** | 选择逐帧或关键帧方式 |
| **Follow Through** | 动作结束后的惯性运动 |
| **Slow In & Slow Out** | 动作加速/减速标注 |
| **Arcs** | 运动轨迹应为弧线 |
| **Secondary Action** | 主体动作外的辅助运动 |
| **Timing** | 每个动作的精确时长 |
| **Exaggeration** | 夸张表演增强表现力 |
| **Solid Drawing** | 角色三维感 |
| **Appeal** | 角色的吸引力和辨识度 |

## 动画分镜检查清单

### 叙事
- [ ] 每个镜头服务故事？
- [ ] 叙事节奏合适？
- [ ] 角色动机清晰？
- [ ] 情感弧线完整？

### 技术
- [ ] 关键姿势标注完整？
- [ ] 动作时间标注精确？
- [ ] 镜头运动清晰？
- [ ] 轴线原则遵守？

### 表演
- [ ] 角色表情传达情绪？
- [ ] 预备动作充足？
- [ ] 动作有变化（避免重复）？
- [ ] 次要动作（头发/衣物）考虑？

### 风格
- [ ] 整体风格统一？
- [ ] 角色比例一致？
- [ ] 色彩方案明确？
- [ ] 背景与角色匹配？
