# 分镜工具与术语参考

## 分镜工具与软件

### 专业分镜软件

| 软件 | 平台 | 特点 | 价格 | 适用人群 |
|------|------|------|------|----------|
| **Storyboarder** | Win/Mac/Linux | 开源免费，功能全面，支持图层 | 免费 | 入门/独立创作者 |
| **StudioBinder** | Web | 在线协作，项目管理，剧本拆分 | 免费+付费 | 团队/专业制作 |
| **FrameForge** | Win/Mac | 3D预演，精确机位模拟，摄影机参数 | 付费 | 专业电影/广告 |
| **Shot Designer** | Win/Mac/iOS | 机位图设计，实时灯光模拟 | 免费+付费 | 导演/摄影指导 |
| **Toon Boom Storyboard Pro** | Win/Mac | 专业动画分镜，与Toon Boom系列无缝衔接 | 付费 | 动画工作室 |
| **Storyboard Fountain** | Mac | 开源，专注于写作+分镜结合 | 免费 | 编剧/导演 |
| **Photoshop** | Win/Mac | 通用工具，模板丰富，图层系统 | 订阅制 | 所有人 |
| **Procreate** | iPad | 手绘手感好，便携，笔刷丰富 | 一次性购买 | 手绘分镜师 |

### AI 辅助分镜工具

| 工具 | 特点 | 适用场景 |
|------|------|----------|
| **Midjourney / DALL-E** | AI生成分镜画面 | 概念设计、Moodboard |
| **Runway ML** | AI视频生成 | 分镜动态预览 |
| **Canva** | 简易分镜模板 | 快速沟通、学生作业 |
| **Figma** | 在线协作，组件复用 | 团队协作分镜 |

### 手绘分镜工具
- **铅笔/炭笔**：快速草图、构思阶段
- **马克笔**：明暗关系、氛围表达
- **数位板（Wacom等）**：数字绘画，可修改
- **iPad + Apple Pencil**：便携数字分镜
- **方格纸**：标准分镜纸替代

### 分镜输出格式

| 格式 | 用途 | 特点 |
|------|------|------|
| **PDF** | 通用查看、打印 | 适合正式提交 |
| **PNG/JPG序列** | 故事板图片集 | 最通用 |
| **PSD/PROCREATE** | 可编辑源文件 | 保留图层信息 |
| **视频（MP4）** | 动态分镜/Animatic | 配合音乐预演 |
| **Excel/CSV** | 镜头数据表 | 项目管理 |
| **FCPXML/ALE** | 剪辑软件导入 | 与后期衔接 |

## 分镜发展历史补充

| 时期 | 里程碑 | 关键人物/作品 |
|------|--------|---------------|
| 1910s | 早期分镜萌芽 | D.W. Griffith《一个国家的诞生》 |
| 1930s | 迪士尼系统化分镜 | 《白雪公主》(1937) |
| 1940s | 希区柯克引入真人电影 | 《迷魂记》《惊魂记》 |
| 1950s | 宽银幕改变构图 | 希区柯克《西北偏北》 |
| 1970s | 斯皮尔伯格精细分镜 | 《大白鲨》《夺宝奇兵》 |
| 1980s | 数字分镜工具出现 | Storyboarder 等软件 |
| 1990s | CG预演兴起 | 《泰坦尼克号》 |
| 2000s | 动态分镜（Animatic）普及 | 皮克斯/梦工厂标准化 |
| 2010s | VR分镜、实时引擎预演 | 虚幻引擎、Unity |
| 2020s | AI辅助分镜生成 | Midjourney、Runway ML |
