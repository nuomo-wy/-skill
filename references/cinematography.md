# 电影摄影与镜头语言

## 景别分类

### 1. 远景 (Extreme Long Shot / ELS)
- **作用**：交代环境、营造氛围、表现规模
- **适用**：开场、转场、大场面
- **情感距离**：观众是旁观者，客观冷静

### 2. 全景 (Long Shot / LS)
- **作用**：展示人物全身及周围环境
- **适用**：人物动作、舞蹈、打斗
- **注意**：人物占画面高度约1/2

### 3. 中景 (Medium Shot / MS)
- **作用**：兼顾人物表情和肢体语言
- **细分**：
  - 中全景 (Medium Long Shot)：膝盖以上
  - 中景：腰部以上
  - 中近景：胸部以上
- **适用**：对话场景最常用

### 4. 近景 (Close Up / CU)
- **作用**：突出表情、传递情绪
- **适用**：情感高潮、关键反应
- **注意**：人物面部占画面主要部分

### 5. 特写 (Extreme Close Up / ECU)
- **作用**：强调细节、制造冲击
- **适用**：眼神、手指、关键道具
- **效果**：强烈的主观感受和情绪张力

## 拍摄角度

### 水平角度
| 角度 | 效果 | 心理暗示 |
|------|------|----------|
| 正面 | 直接、正式 | 平等交流 |
| 侧面 | 动感、轮廓 | 旁观、神秘 |
| 斜侧 | 立体感强 | 自然、日常 |
| 背面 | 悬念、未知 | 代入感强 |

### 垂直角度
| 角度 | 效果 | 心理暗示 |
|------|------|----------|
| 俯拍 | 渺小、弱势 | 上帝视角、压抑 |
| 平拍 | 平等、客观 | 中立、自然 |
| 仰拍 | 高大、强势 | 敬畏、威胁 |

## 镜头运动

### 固定镜头 (Static Shot)
- **特点**：稳定、客观、强调构图
- **适用**：对话、静物、需要沉淀情绪的场景

### 推镜头 (Push In / Dolly In)
- **作用**：引导注意力、强调细节、情绪递进
- **节奏**：慢推=思考/情感积累，快推=冲击/发现

### 拉镜头 (Pull Out / Dolly Out)
- **作用**：展示环境、制造疏离、揭示真相
- **效果**：从局部到整体，信息逐步展开

### 摇镜头 (Pan)
- **水平摇**：展示空间、跟随移动主体
- **速度**：慢摇=优雅/搜索，快摇=紧张/冲击

### 移镜头 (Truck/Dolly)
- **横移**：平行跟随，保持主体位置
- **纵深移**：空间穿越感

### 跟镜头 (Tracking Shot)
- **作用**：跟随主体移动，保持相对位置
- **效果**：强烈的临场感和参与感

### 升降镜头 (Crane/Jib)
- **作用**：空间转换、情绪升华
- **效果**：宏大感、戏剧性

### 手持/斯坦尼康 (Handheld/Steadicam)
- **手持**：真实感、紧张感、纪录片风格
- **斯坦尼康**：流畅中带呼吸感

## 构图法则

### 三分法则 (Rule of Thirds)
- 画面横竖各分三等分
- 重要元素放在交点或线上
- 最基础也最实用的构图原则

### 黄金分割
- 比例约 1:1.618
- 视觉上最舒适的比例关系

### 对称构图
- **效果**：稳定、正式、庄严
- **适用**：建筑、仪式、对比场景

### 引导线 (Leading Lines)
- 利用画面中的线条引导视线
- 道路、栏杆、建筑边缘等

### 框架构图 (Frame within Frame)
- 利用门窗、镜子等形成画中画
- **效果**：强调主体、增加层次

### 留白 (Negative Space)
- **作用**：呼吸感、孤独感、想象空间
- **适用**：文艺片、情绪表达

## 光线运用

### 光位
| 光位 | 特点 | 情绪 |
|------|------|------|
| 顺光 | 明亮、清晰 | 明快、日常 |
| 侧光 | 立体感强 | 戏剧、神秘 |
| 逆光 | 轮廓、剪影 | 梦幻、孤独 |
| 顶光 | 特殊效果 | 恐怖、压抑 |
| 底光 | 非自然 | 恐怖、诡异 |

### 光质
- **硬光**：明暗对比强烈，戏剧性
- **软光**：过渡柔和，自然唯美

### 三点布光
1. **主光 (Key Light)**：主要光源，塑造主体
2. **辅光 (Fill Light)**：填补阴影，控制对比
3. **轮廓光 (Rim Light)**：分离主体与背景

## 色彩与影调

### 影调
- **高调**：明亮、轻快、纯净
- **低调**：阴暗、沉重、神秘
- **中间调**：自然、平衡

### 色调情绪
| 色调 | 情绪联想 |
|------|----------|
| 暖色调 | 温暖、怀旧、激情 |
| 冷色调 | 冷静、疏离、未来 |
| 高饱和 | 活力、夸张、戏剧 |
| 低饱和 | 压抑、现实、文艺 |
| 黑白 | 经典、严肃、永恒 |

## 镜头时长与节奏

### 长镜头
- **定义**：通常 >10秒
- **效果**：真实时间流逝、沉浸感
- **大师**：侯孝贤、贝拉·塔尔

### 短镜头
- **定义**：通常 <3秒
- **效果**：紧张、快节奏、信息密集
- **适用**：动作片、MV、广告

### 剪辑节奏
- **快剪**：兴奋、混乱、现代感
- **慢剪**：沉思、优雅、经典
- **跳切**：不安、时间压缩、风格化

## 运镜速度与情绪

| 速度 | 效果 | 适用场景 |
|------|------|----------|
| **极慢** | 沉思、时间凝滞、诗意 | 文艺片、情感高潮、结尾 |
| **缓慢** | 优雅、思考、情感积累 | 对话、观察、氛围建立 |
| **正常** | 自然、叙事流畅 | 日常场景、叙事推进 |
| **快速** | 紧张、兴奋、混乱 | 动作、追车、冲突 |
| **极快** | 冲击、震惊、风格化 | 暴力、爆炸、闪回 |

**关键规律**：速度变化本身也是叙事工具——匀速营造舒适，变速制造不安。

## 复合运镜

| 组合 | 效果 | 典型应用 |
|------|------|----------|
| **推+摇** | 引导视线同时展示环境 | 角色走向目标 |
| **移+升** | 空间转换、情绪升华 | 史诗性揭示 |
| **跟+拉** | 跟随同时揭示更大场景 | 离开角色、展示全貌 |
| **手持+推** | 紧张中的情绪递进 | 恐怖、审讯 |
| **升+拉** | 从特写直接到全景 | 情绪释放、场景结束 |
| **摇+推** | 先展示空间再聚焦主体 | 展示→发现 |
| **移+跟** | 保持距离跟随 | 平行叙事、长镜头 |

## 高级构图技巧

| 技巧 | 效果 | 适用场景 |
|------|------|----------|
| **层次构图** | 前景/中景/背景三层，增加深度 | 室内、建筑、对话 |
| **对角线构图** | 动感、不稳定、力量 | 动作、冲突 |
| **三角形构图** | 稳定、力量感 | 领导者、英雄 |
| **圆形构图** | 完整、循环、封闭 | 关系、宿命 |

| **打破构图** | 不安、冲突、现代感 | 心理不稳定、恐怖 |

## 经典镜头组合

### 正反打 (Shot/Reverse Shot)
- **用途**：对话场景标准拍法
- **要点**：轴线原则，视线匹配

### 主镜头法 (Master Scene)
1. 先拍全景建立空间关系
2. 再拍中景、近景补充细节

### 过肩镜头 (Over the Shoulder)
- **效果**：对话的参与感
- **注意**：肩膀占画面约1/3

### 插入镜头 (Insert Shot)
- **用途**：强调细节、道具、反应
- **时机**：在动作的关键节点

## 轴线原则

### 180度规则
- 在假想轴线一侧拍摄
- 保持空间方向一致
- 越轴需明确动机或过渡

### 常见轴线
- **视线轴**：两人对视的连线
- **运动轴**：物体移动的方向线
- **关系轴**：场景中元素的关系线
