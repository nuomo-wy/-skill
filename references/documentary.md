# 纪录片分镜完全指南

## 纪录片分镜特点

### 与虚构电影的区别
| 维度 | 纪录片 | 虚构电影 |
|------|--------|----------|
| 内容 | 真实事件/人物 | 虚构故事 |
| 分镜时机 | 拍摄前/后都有 | 主要拍摄前 |
| 控制程度 | 低（事件不可控） | 高（可要求重拍） |
| 创作自由度 | 发现式 | 设计式 |
| 叙事结构 | 可后建 | 预设 |

### 纪录片分镜核心原则
1. **真实优先**：不为了分镜牺牲真实
2. **观察为主**：减少干预
3. **灵活性**：随时准备捕捉决定性瞬间
4. **叙事构建**：后期叙事与前期拍摄同等重要

## 纪录片分镜类型

### 1. 观察式纪录片 (Observational)
**特点**：
- 纯粹观察，少/无旁白
- 长镜头为主
- 摄影机尽量不干预
- 建立身临其境感

**代表案例**：

**Frederick Wiseman 作品**
```
《Titicut Follies》(1967)
《High School》(1968)
特点：
- 极简干预
- 长时间观察
- 日常中发现意义
```

**分镜策略**：
- 固定镜头，观察等待
- 斯坦尼康跟随自然移动
- 录音质量比画面更重要
- 预判+等待

---

**D.A. Pennebaker 作品**
```
《Dont Look Back》(1967) - 鲍勃·迪伦
《Monterey Pop》(1968)
特点：
- 音乐/文化事件记录
- 摄影机作为旁观者
- 意外/非编排时刻捕捉
```

---

### 2. 参与式纪录片 (Participatory)
**特点**：
- 导演/拍摄者入镜
- 直接与被摄者互动
- 主观视角明确
- 个人探索主题

**代表案例**：

**Michael Moore 作品**
```
《Roger & Me》(1989)
《Bowling for Columbine》(2002)
《Fahrenheit 9/11》(2004)
特点：
- 导演高度介入
- 观点明确
- 辩论/说服导向
```

**分镜策略**：
- 导演入画，直接提问
- 采访场景多机位
- 反打镜头构建对话
- 插入档案/新闻素材

---

**Werner Herzog 作品**
```
《Grizzly Man》(2005)
《Into the Abyss》(2011)
特点：
- 导演声音明确
- 哲学/诗意探索
- 采访+实地拍摄
```

---

### 3. 述评式纪录片 (Expository)
**特点**：
- 强叙事旁白
- 画面配合文字/解说
- 观点明确
- 教育/信息导向

**代表案例**：

**BBC 自然纪录片**
```
《地球脉动》(Planet Earth)
《蓝色星球》(Blue Planet)
特点：
- 大卫·爱登堡旁白
- 精美的野生动物摄影
- 叙事结构清晰
```

**分镜策略**：
- B-Roll 丰富
- 采访作为观点支撑
- 场景重现/动画解释
- 旁白引导叙事

---

**Errol Morris 作品**
```
《细细的蓝线》(The Thin Blue Line, 1988)
《标准流程》(Standard Operating Procedure, 2008)
特点：
- 采访重建事件
- 档案素材+新拍摄
- 侦探式叙事
```

---

### 4. 反身式纪录片 (Reflexive)
**特点**：
- 摄影机本身是主题
- 质疑纪录片真实性
- 元叙事
- 观众参与构建

**代表案例**：

**Jean Rouch - Chronique d'un été (1961)**
```
"真实性如何？"
电影-verité运动开创
被摄者与摄影机互动被展示
```

**分镜策略**：
- 展示采访过程
- 被摄者对镜反应
- 摄影机/拍摄者入画
- 拒绝隐藏"幕后"

---

**Forensic Architecture**
```
调查新闻+3D重建
《Killing in Umm al-Hiran》(2017)
特点：
- 事件重建
- 多角度证据
- 调查过程展示
```

## 纪录片拍摄分镜

### 事前分镜（设计阶段）
```
确定主题 → 研究 → 预想镜头 → 场景列表
```

**预想镜头类型**：
- B-Roll（插入镜头）清单
- 采访问题设计
- 场景建立镜头需求
- 重要时刻预期

---

### 现场分镜（灵活应对）
```
观察 → 等待 → 捕捉 → 评估
```

**观察阶段**：
- 熟悉环境
- 识别关键人物
- 预判可能发生的事
- 确定机位

**等待阶段**：
- 设定镜头
- 调整参数
- 保持耐心
- 录音待命

**捕捉阶段**：
- 决定性瞬间
- 关键反应
- 意外事件
- 情感高潮

## 纪录片分镜要素

### 1. 建立镜头 (Establishing Shot)
```markdown
用途：建立场景/人物/时间

类型：
- 环境全景：展示地点
- 人物全身：介绍角色
- 时间标记：时钟/日历
- 季节暗示：自然/天气

示例：《华氏911》农场建立恐怖训练营
```

---

### 2. 采访镜头 (Interview Shot)
```markdown
标准设置：
- 45度侧面
- 深色背景
- 眼神线与相机一致
- 适度特写

反打角度：
- 采访者入画或画外音
- 被访者看向镜头外
- 视线匹配构建对话感
```

---

### 3. B-Roll (插入镜头)
```markdown
定义：补充主镜头的内容镜头

类型：
- 环境：展示地点
- 动作：描述活动
- 细节：强调重点
- 过渡：场景连接
- 情感：反应镜头

示例：
主镜头 = 采访医生说"手术很成功"
B-Roll = 手术室全景/器械/医生表情/走廊
```

---

### 4. 观察镜头 (Observational Shot)
```markdown
特点：
- 长镜头：30秒以上
- 固定或缓慢移动
- 无旁白/无干预
- 真实时间流逝

目的：
- 建立临场感
- 让观众自己判断
- 捕捉意外时刻
```

---

### 5. POV 镜头 (主观镜头)
```markdown
用途：观众代入角色视角

类型：
- 角色行走视角
- 第一人称叙述
- 角色所看之物
- 角色所感之物

示例：《寻找小脚八号》跟随棒球运动员
```

## 纪录片叙事结构

### 经典三幕结构
```
第一幕：建立主题
- 引入问题/人物
- 建立观众兴趣
- 预览冲突

第二幕：发展
- 深入主题
- 展示多面性
- 建立复杂性
- 升级冲突

第三幕：解决
- 高潮时刻
- 反思/开放式结局
- 行动召唤
```

---

### 章节式结构
```markdown
将主题分解为多个部分
每个章节有独立叙事弧

示例：
《731部队》分为：
- 历史背景
- 受害者故事
- 加害者视角
- 遗留影响
- 国际反思
```

---

### 时间线结构
```markdown
按时间顺序叙述
适合：历史事件、传记、成长故事

类型：
- 线性：过去→现在
- 倒叙：现在→过去→现在
- 交叉：多条时间线并行

示例：《制造业的真相》按生产流程拍摄
```

---

### 主题并列结构
```markdown
围绕核心主题
展示多个相关但独立的故事

示例：《监禁文化》
- 监狱系统
- 家庭影响
- 社会成本
- 替代方案
```

## 纪录片分镜技巧

### 1. 预判与等待
```markdown
技巧：
- 研究被摄者和环境
- 提前设置机位
- 预测可能发生的事
- 保持警觉
- 持续录音

案例： Maysles兄弟的"真实时刻"哲学
```

---

### 2. 最小干预原则
```markdown
技巧：
- 摄影机不要引导事件
- 保持一定距离
- 允许自然发展
- 观察>提问

目的：保持真实性
```

---

### 3. 声音优先
```markdown
技巧：
- 纪录片声音比画面更重要
- 同步录音优先
- 环境音建立空间
- 采访时高质量收音

案例：《华氏911》伊拉克战争场景依赖声音
```

---

### 4. 意外捕捉
```markdown
技巧：
- 保持摄影机运转
- 意外时刻往往最重要
- 不要因为"没准备好"错过
- B卷意识

案例：D.A. Pennebaker 捕捉 Bob Dylan 的意外时刻
```

## 不同主题的分镜策略

### 政治/社会议题
**特点**：
- 观点明确
- 证据展示
- 多方声音
- 行动导向

**分镜要点**：
- 采访关键人物
- 档案素材丰富
- 现场拍摄真实
- 数据可视化

---

### 自然/野生动物
**特点**：
- 精美画面
- 耐心等待
- 专业设备
- 科学叙事

**分镜要点**：
- 远距离长焦
- 动作预判
- 栖息地建立
- 特写细节

---

### 人物传记
**特点**：
- 采访为核心
- 档案素材
- 场景重建
- 情感叙事

**分镜要点**：
- 深度采访
- 家庭/工作环境
- 关键物件特写
- 情感时刻捕捉

---

### 历史事件
**特点**：
- 档案为主
- 重建场景
- 多方视角
- 事实核查

**分镜要点**：
- 历史影像/照片
- 场景重建
- 采访幸存者/见证人
- 地图/时间线动画

---

### 音乐/艺术
**特点**：
- 表演为核心
- 幕后故事
- 创作过程
- 美学展示

**分镜要点**：
- 多机位表演
- 幕后花絮
- 创作过程
- 艺术家采访

## 纪录片分镜检查清单

### 拍摄前
- [ ] 主题清晰
- [ ] 研究充分
- [ ] 关键人物已联系
- [ ] 设备准备完整
- [ ] B-Roll 清单准备

### 拍摄中
- [ ] 建立镜头充足
- [ ] 同步录音良好
- [ ] 采访画面+反打
- [ ] 环境音收录
- [ ] 意外时刻捕捉
- [ ] 多角度覆盖

### 后期素材
- [ ] 素材标注清晰
- [ ] 时长记录
- [ ] 重要时刻标记
- [ ] 潜在故事线记录

## 纪录片分镜案例分析

### 《华氏911》(2004) - Michael Moore
**分镜策略**：
- 个人视角贯穿
- 档案素材大量运用
- 采访+实地拍摄结合
- 争议性叙事结构

**视觉特点**：
- 采访：简洁，直接
- 档案：处理后融入叙事
- 实地：Moore 本人入画
- 讽刺性并列

---

### 《寻找小脚八号》(1999) - Errol Morris
**分镜策略**：
- 斯坦尼康跟拍运动员
- 采访构建叙事
- 档案/照片补充
- 开放式结局

**视觉特点**：
- 跟拍长镜头建立节奏
- 采访场景极简
- 运动动作分解
- 人物内心探索

---

### 《难以忽视的真相》(2006) - Davis Guggenheim
**分镜策略**：
- Gore 演讲为核心
- 数据可视化
- 极端天气画面
- 个人情感连接

**视觉特点**：
- 简洁背景 Gore 采访
- 科学数据动画化
- 极端事件档案
- 家庭/个人时刻

---

### 《脸庞，村庄》(2017) - Agnès Varda & JR
**分镜策略**：
- 移动摄影车
- 村庄探索
- 人物肖像放大
- 诗意叙事

**视觉特点**：
- JR 卡车摄影车作为移动工作室
- 人物肖像巨大化
- 法国乡村风景
- 老年/边缘人群关注
