# 广告分镜专项指南

## 广告分镜特点

### 时间压力
- 15秒、30秒、60秒是标准时长
- 每个镜头必须高效传递信息
- 平均 2-4秒/镜头

### 核心任务
1. **吸引注意** (前3秒)
2. **传递信息** (产品卖点)
3. **情感连接** (品牌调性)
4. **行动召唤** (CTA)

## 广告类型分镜策略

### 1. 产品展示型
**结构**：问题 → 解决方案 → 产品 → 利益点

| 阶段 | 镜头策略 | 时长占比 |
|------|----------|----------|
| 钩子 | 冲击画面/问题场景 | 10% |
| 引入 | 产品亮相，视觉焦点 | 20% |
| 展示 | 功能演示，多角度 | 40% |
| 升华 | 使用场景，情感价值 | 20% |
| 结尾 | 品牌+CTA | 10% |

**分镜要点**：
- 产品必须在3秒内出现或暗示
- 特写镜头突出产品质感
- 动态展示优于静态

### 2. 故事情感型
**结构**：人物 → 冲突 → 转折 → 解决 → 品牌关联

| 阶段 | 镜头策略 |
|------|----------|
| 铺垫 | 中景建立人物关系 |
| 发展 | 近景捕捉情绪变化 |
| 高潮 | 特写/慢动作强化 |
| 解决 | 产品自然融入 |
| 回味 | 留白镜头，品牌露出 |

**分镜要点**：
- 前5秒必须建立情感连接
- 产品出现要自然，不突兀
- 结尾留品牌记忆点

### 3. 生活方式型
**结构**：理想场景 → 产品融入 → 价值观传递

**分镜要点**：
- 大景别展示理想环境
- 中景展示人物状态
- 产品作为"道具"自然出现
- 色调统一，风格化明显

### 4. 证言/演示型
**结构**：信任建立 → 问题呈现 → 解决方案 → 效果证明

**分镜要点**：
- 真人出镜增加可信度
- 前后对比镜头
- 数据可视化展示
- 真实场景拍摄

## 广告分镜技巧

### 开场钩子 (The Hook)
**前3秒决定生死**

有效钩子类型：
1. **视觉冲击**：非常规画面、强烈色彩
2. **问题共鸣**：直击痛点的问题
3. **悬念设置**：引发好奇的画面
4. **情绪调动**：笑容、惊讶、温暖

### 产品亮相 (The Reveal)
**让产品成为明星**

- **时机**：不宜过早（需要铺垫），不宜过晚（信息不足）
- **方式**：
  - 直接亮相：产品占据画面中心
  - 悬念揭晓：从局部到整体
  - 场景融入：产品自然出现在使用场景
- **光线**：产品光要精致，突出质感

### 信息层级
**观众能记住的有限**

一个30秒广告，核心信息不超过3个：
1. 主信息（必须记住）
2. 次信息（加深印象）
3. 辅助信息（锦上添花）

### 品牌记忆 (Brand Recall)
**结尾必须强化**

- 品牌Logo展示 ≥ 2秒
- 品牌色彩贯穿
- 品牌音乐/音效
- Slogan配合画面

## 分镜节奏控制

### 15秒广告节奏
```
0-2s   钩子
2-5s   问题/场景
5-10s  产品+核心卖点
10-13s 利益点/情感升华
13-15s 品牌+CTA
```

### 30秒广告节奏
```
0-3s   钩子
3-8s   情境建立
8-18s  产品展示+卖点
18-25s 使用场景/情感连接
25-28s 品牌强化
28-30s CTA
```

### 60秒广告节奏
- 可以容纳更完整的故事弧
- 允许情绪铺垫和转折
- 多个卖点可以分层展示
- 品牌可以多次露出

## 常见广告分镜模板

### 模板A：问题解决型
| 镜号 | 景别 | 内容 | 时长 |
|------|------|------|------|
| 1 | 中景 | 人物遇到困扰 | 3s |
| 2 | 特写 | 困扰细节 | 2s |
| 3 | 近景 | 发现产品 | 2s |
| 4 | 特写 | 产品特写 | 3s |
| 5 | 中景 | 使用产品 | 4s |
| 6 | 近景 | 问题解决，满意表情 | 3s |
| 7 | 特写 | 产品+品牌 | 3s |
| 8 | 全黑 | Slogan+Logo | 2s |

### 模板B：情感故事型
| 镜号 | 景别 | 内容 | 时长 |
|------|------|------|------|
| 1 | 远景 | 环境，时间感 | 3s |
| 2 | 中景 | 人物活动 | 4s |
| 3 | 近景 | 情感互动 | 5s |
| 4 | 特写 | 关键情感瞬间 | 3s |
| 5 | 中景 | 产品自然出现 | 4s |
| 6 | 近景 | 使用产品，情感升华 | 5s |
| 7 | 远景 | 美好场景 | 3s |
| 8 | 特写 | 品牌Logo | 2s |

### 模板C：快节奏剪辑型
- 镜头时长：1-2秒
- 快速切换不同场景
- 音乐驱动剪辑点
- 产品多角度闪现
- 结尾定格品牌

## 广告分镜检查清单

- [ ] 前3秒能抓住注意力吗？
- [ ] 产品出现时机合适吗？
- [ ] 核心信息清晰吗？
- [ ] 每个镜头都有必要吗？
- [ ] 品牌露出足够吗？
- [ ] CTA明确吗？
- [ ] 时长控制准确吗？
- [ ] 目标受众会共鸣吗？
