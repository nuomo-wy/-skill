# 声音叙事设计指南

## 声音在分镜中的地位

**声音不是画面的附属品**——它是叙事的另一半。

- 画面告诉观众"发生了什么"
- 声音告诉观众"这意味着什么"
- 声画结合告诉观众"应该如何感受"

**核心理念**：在分镜阶段就规划声音，而不是后期补上。

---

## 声音五大要素

| 要素 | 定义 | 分镜标注 |
|------|------|----------|
| **对白 (Dialogue)** | 角色说话 | 逐字标注 |
| **旁白 (Voice Over)** | 画外叙述 | 逐字标注+语气 |
| **音效 (SFX)** | 非音乐非人声的声音 | 描述性标注 |
| **环境音 (Ambient)** | 场景氛围声音 | 简要标注 |
| **音乐 (Music)** | 背景音乐/配乐 | 风格+情绪+节奏 |

## 声画关系

### 六种声画关系

| 关系 | 原理 | 效果 | 示例 |
|------|------|------|------|
| **声画同步** | 声音与画面内容一致 | 强化、确认 | 看到打雷→听到雷声 |
| **声画分离** | 声音与画面来源不同 | 扩展空间、增加信息 | 看到主角沉默→听到旁白 |
| **声画对位** | 声音与画面情绪相反 | 讽刺、反差、深度 | 暴力画面+古典音乐（《发条橙》） |
| **声画平行** | 声音与画面各自发展但主题相关 | 丰富层次 | 看到城市+听到访谈 |
| **声画替代** | 声音替代画面表现 | 悬念、想象 | 只听到枪声，不见开枪 |
| **静默** | 声音突然消失 | 冲击、内心、时间凝滞 | 爆炸后的短暂无声 |

### 何时用声画对位

声画对位是最强大的声音叙事工具，但也最危险——用不好就是失误。

**有效场景**：
- 暴力场景配轻快音乐 → 暴力的荒诞性
- 悲伤画面配欢快音乐 → 角色的自我欺骗
- 日常场景配紧张音乐 → 暗示危机即将到来
- 高潮场景配寂静 → 内心的震惊

**失败场景**：
- 随意使用，没有叙事动机
- 观众困惑"这是故意的还是失误"
- 声画情绪差距太小，效果不明显

## 对白设计

### 分镜中的对白标注

**标准格式**：
```
角色A（语气）："台词内容"
```

**示例**：
```
主角（低声，犹豫）："我不知道……该不该去。"
```

### 对白与景别的关系

| 景别 | 对白处理 | 原因 |
|------|----------|------|
| 远景/全景 | 尽量避免对白 | 看不到嘴型，不自然 |
| 中景 | 标准对话景别 | 最自然的对话画幅 |
| 近景/特写 | 对白可省略，表情代替 | 面部表情比语言更有力 |
| 画外 | 旁白/回忆/电话 | 声音与画面分离 |

### 对白节奏

| 技巧 | 效果 | 示例 |
|------|------|------|
| 打断 | 紧张、冲突 | A说话时B插嘴 |
| 沉默 | 思考、震撼 | 问完问题后的3秒静默 |
| 重叠 | 混乱、兴奋 | 多人同时说话 |
| 重复 | 强调、执念 | 反复说同一句话 |
| 渐弱 | 放弃、疲惫 | 声音越来越小 |

## 旁白设计

### 旁白类型

| 类型 | 效果 | 适用 |
|------|------|------|
| **全知旁白** | 客观叙述，上帝视角 | 纪录片、史诗 |
| **角色旁白** | 主观感受，内心世界 | 剧情片、回忆 |
| **第二人称** | 直接对话观众 | 广告、互动视频 |
| **反讽旁白** | 与画面形成反差 | 黑色幽默 |

### 旁白节奏控制

**基础规则**：
- 语速：中文约 4字/秒（正常）、3字/秒（慢速）、5字/秒（快速）
- 停顿：逗号0.3秒，句号0.5秒，段落1-2秒
- 旁白不要塞满——画面需要呼吸

**旁白密度参考**：
| 类型 | 旁白密度 | 说明 |
|------|----------|------|
| 纪录片 | 60-80% | 旁白驱动叙事 |
| 广告 | 40-60% | 旁白+画面各半 |
| 剧情片 | 10-20% | 旁白仅用于关键内心独白 |
| MV | 0-10% | 极少旁白，音乐为主 |
| 短视频 | 50-70% | 旁白是信息主体 |

## 音效设计

### 音效分类

| 分类 | 作用 | 分镜标注 |
|------|------|----------|
| **动作音效** | 强化动作感知 | 关门声、脚步声、拳头声 |
| **环境音效** | 建立空间感 | 雨声、交通声、市场声 |
| **情绪音效** | 心理暗示 | 心跳、耳鸣、低频震动 |
| **转场音效** | 场景切换辅助 | Whoosh、上升音、下降音 |
| **设计音效** | 风格化、超现实 | 夸张的漫画音、电子音 |

### 动作音效强化原理

**为什么电影里的声音比现实更响？**

因为观众需要**感知**而非**记录**——
- 现实中关门声很小 → 电影里放大，让观众"感觉"到力度
- 现实中不会注意脚步 → 电影里强调，建立节奏
- 现实中心脏不会这样跳 → 电影里加入心跳，制造紧张

### 分镜中的音效标注

**简洁版**：
```
声音: 关门声 + 脚步声（渐近）
```

**详细版**：
```
声音设计:
  主音效: 关门声（沉重、回响）
  环境音: 走廊的空旷回声
  情绪音: 低频嗡鸣（暗示危险）
  节奏: 关门声→1秒静默→脚步声开始
```

## 环境音设计

### 环境音的三层结构

| 层级 | 内容 | 作用 |
|------|------|------|
| **底色层** | 持续的环境声（Room Tone） | 建立空间基础 |
| **细节层** | 间歇的具体声音 | 增加真实感 |
| **叙事层** | 与剧情相关的环境声 | 推动叙事/暗示 |

**示例——咖啡馆场景**：
```
底色层: 咖啡机嗡鸣 + 隐约的人声嘈杂
细节层: 杯碟碰撞 + 拉链声 + 门铃声
叙事层: 雨声渐大（暗示角色心情变化）
```

### 环境音与情绪

| 环境 | 安静版情绪 | 嘈杂版情绪 |
|------|-----------|-----------|
| 城市 | 凌晨的孤独 | 白天的活力/焦虑 |
| 森林 | 宁静、神秘 | 不安、潜伏 |
| 海边 | 平静、永恒 | 风暴、力量 |
| 办公室 | 空洞、加班 | 忙碌、压力 |
| 家中 | 温馨、安全 | 争吵、混乱 |

## 音乐设计

### 音乐在分镜中的标注

| 标注项 | 说明 | 示例 |
|--------|------|------|
| **风格** | 音乐类型 | 钢琴独奏/电子/弦乐/摇滚 |
| **情绪** | 情感方向 | 温暖/紧张/忧伤/激昂 |
| **节奏** | 速度+节拍 | 慢板60BPM/快板120BPM |
| **动态** | 音量变化 | 渐强/渐弱/突强/突弱 |
| **起止** | 进入和退出时机 | 镜头3起，镜头8收 |

### 音乐节奏与剪辑节奏

**黄金法则**：剪辑点与音乐节拍同步，观众感觉"顺畅"。

| 同步程度 | 效果 |
|----------|------|
| 精确踩拍 | 最流畅，专业感 |
| 接近踩拍 | 自然，不刻意 |
| 故意错拍 | 不安，风格化 |
| 完全不同步 | 混乱，业余 |

**分镜标注**：
```
音乐: 钢琴+弦乐，温暖，72BPM
节奏提示: 镜头切换踩1、3拍
动态: 镜头5渐强，镜头9高潮
```

### 音乐起止设计

| 音乐进入方式 | 效果 | 适用 |
|-------------|------|------|
| 渐入 (Fade In) | 自然、温柔 | 情感场景 |
| 硬入 (Hard In) | 冲击、惊喜 | 转折、高潮 |
| 声音桥引入 | 流畅、连接 | 场景转换 |
| 从对白后进入 | 情绪释放 | 对话结束后的感悟 |

| 音乐退出方式 | 效果 | 适用 |
|-------------|------|------|
| 渐出 (Fade Out) | 自然结束 | 段落收束 |
| 硬切 (Hard Out) | 突兀、冲击 | 惊吓、打断 |
| 降到最低 | 保留氛围 | 进入对话 |
| 消失于环境音 | 无缝融入现实 | 回到现实 |

## 静默的力量

**静默≠无声**——它是声音叙事的最强武器。

### 静默类型

| 类型 | 定义 | 效果 |
|------|------|------|
| **绝对静默** | 所有声音消失 | 震惊、内心、超现实 |
| **相对静默** | 音乐消失，只留环境音 | 回到现实、沉淀 |
| **预期静默** | 观众预期有声音但没有 | 悬念、不安 |
| **后震静默** | 巨大声音之后的静默 | 余震、创伤 |

### 何时使用静默

- **情感高潮**：最强烈的情感不需要配乐
- **死亡/失去**：语言的尽头
- **重大决定**：内心独白的空间
- **震惊瞬间**：爆炸后的短暂无声
- **诗意留白**：给观众思考空间

**分镜标注**：
```
声音: [静默2秒] — 所有声音消失
备注: 爆炸后的耳鸣效果，画面微晃
```

## 声音转场（与 transition.md 联动）

### J-Cut 设计要点
- 声音提前量：1-3秒（电影）、0.5-2秒（广告/短视频）
- 提前的声音要与下一个场景有明确关联
- 不适合：声音与下场景无关（观众会困惑）

### L-Cut 设计要点
- 声音延续量：1-5秒
- 延续的声音要让当前画面产生新含义
- 经典用法：上一个场景的对白延续到主角独处的画面

### 声音桥设计要点
- 选择贯穿多个场景的统一声音元素
- 可以是：旁白、音乐、环境音、音效
- 声音桥下可以快速切换多个画面（蒙太奇）

## 分镜声音标注格式

### 简洁版（表格分镜）
```
| 声音 | 说明 |
|------|------|
| 对白:角色A "台词" | 语气标注 |
| SFX: 关门声 | 动作音效 |
| BGM: 钢琴渐入 | 风格+动态 |
```

### 详细版（剧本分镜）
```
声音设计:
  对白:
    主角（低声）: "我必须这样做。"

  音效:
    主: 风声（从左到右渐强）
    辅: 远处钟声（2秒后）
    情绪: 低频嗡鸣

  音乐:
    BGM: 弦乐渐强，从pp到ff
    进入: 旁白结束后2秒
    高潮: 镜头9

  静默:
    镜头10-11之间 [静默1.5秒]
```

### 短视频版（紧凑标注）
```
声: 旁白"你知道吗？" + 悬疑音效
```

## 声音设计检查清单

### 基础完整性
- [ ] 每个镜头都有声音标注？
- [ ] 对白逐字标注？
- [ ] 环境音与场景匹配？
- [ ] 音乐风格与情绪一致？

### 叙事功能
- [ ] 声音是否服务于叙事？
- [ ] 有没有声画对位的机会？
- [ ] 转场是否需要J-cut/L-cut？
- [ ] 关键时刻是否用了静默？

### 节奏控制
- [ ] 旁白密度合适？
- [ ] 音乐与剪辑节奏同步？
- [ ] 声音有层次（不全是一个音量）？
- [ ] 有"呼吸感"（不是持续满载）？

### 技术可行性
- [ ] 声音设计在预算内可实现？
- [ ] 是否需要特殊录音？
- [ ] 音乐版权是否解决？
- [ ] 环境音是否需要实地录制？
